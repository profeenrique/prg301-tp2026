En esta unidad trabajaremos con los elementos fundamentales de la programación en Python. Revisaremos cómo almacenar información, cómo operar con ella y cómo interactuar mediante entrada y salida de datos.

## Objetivos de aprendizaje

Al finalizar esta unidad, serás capaz de:

- Utilizar variables para almacenar información en programas simples.
- Reconocer tipos de datos básicos y aplicar conversiones entre ellos cuando sea necesario.
- Aplicar operadores aritméticos, de comparación y lógicos en expresiones simples.
- Utilizar `print()` e `input()` para mostrar información y capturar datos desde el teclado.

## Requisitos previos

- Contar con Python instalado y operativo.
- Poder crear, guardar y ejecutar archivos `.py`.
- Manejar de forma básica Visual Studio Code y la terminal.
