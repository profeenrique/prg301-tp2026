En esta unidad se abordarán las bases con las que comienza a construirse cualquier programa en Python. Aquí se trabajará con variables, tipos de datos, operadores y mecanismos básicos de interacción, es decir, con los elementos que permiten almacenar información, transformarla y comunicar resultados.

Más que un conjunto de definiciones aisladas, esta unidad introduce una forma inicial de pensar en programación: cómo representar datos, cómo operar con ellos y cómo hacer que un programa pueda recibir información y responder a ella.

## Objetivos de aprendizaje

Al finalizar esta unidad, serás capaz de:

- Utilizar variables para representar y almacenar información en programas simples
- Reconocer tipos de datos básicos en Python y aplicar conversiones cuando el programa lo requiera
- Usar operadores aritméticos, de comparación y lógicos para construir expresiones
- Emplear `print()` e `input()` para mostrar información y capturar datos desde el teclado

## Requisitos previos

Para trabajar esta unidad, conviene que puedas:

- Crear, guardar y ejecutar archivos `.py`
- Usar de forma básica Visual Studio Code y la terminal
- Contar con Python instalado y funcionando correctamente en tu equipo
