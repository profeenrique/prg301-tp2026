En esta unidad prepararemos nuestro entorno de trabajo y escribiremos nuestras primeras líneas de código. Instalaremos Python, configuraremos Visual Studio Code y revisaremos la sintaxis básica del lenguaje para comenzar a ejecutar scripts simples de forma ordenada.

## Objetivos de aprendizaje

Al finalizar esta unidad, serás capaz de:

- Configurar un entorno de desarrollo local con Python y Visual Studio Code.
- Ejecutar scripts simples desde la terminal del sistema.
- Escribir un primer programa en Python.
- Probar programas simples y corregir errores comunes de sintaxis y ejecución.

## Requisitos previos

- Manejo básico del sistema operativo, especialmente creación de carpetas y navegación de archivos.
- Acceso a un equipo con permisos suficientes para instalar o ejecutar el software requerido.

## Aplicación de la unidad

Al finalizar esta unidad, contarás con un entorno de trabajo funcional para comenzar a programar en Python. Podrás ejecutar scripts simples, interpretar la salida en la terminal y realizar correcciones básicas ante errores comunes.
