En esta unidad se construirá la base desde la cual comenzará el trabajo con Python. Antes de resolver problemas más complejos, conviene preparar con cuidado el entorno, comprender cómo ejecutar un programa y familiarizarse con las primeras reglas de escritura del lenguaje.

Aquí se abordará un primer acercamiento tanto a Python como al espacio de trabajo desde el que se programará. Esto incluye la instalación del lenguaje, la configuración de Visual Studio Code, el uso inicial de la terminal y la escritura de programas simples que permitan observar cómo responde el intérprete.

## Objetivos de aprendizaje

Al finalizar esta unidad, serás capaz de:

- Preparar un entorno de desarrollo local para trabajar con Python y Visual Studio Code
- Crear, guardar y ejecutar scripts simples desde la terminal
- Escribir y probar un primer programa en Python
- Reconocer y corregir errores básicos de sintaxis y ejecución
- Comprender el papel del intérprete, del editor y de la terminal en el flujo de trabajo inicial

## Requisitos previos

Para trabajar esta unidad, conviene que puedas:

- Manejar de forma básica el sistema operativo
- Crear carpetas, abrir archivos y navegar entre ubicaciones del equipo
- Contar con un equipo que permita instalar o ejecutar el software necesario

## Aplicación de la unidad

Al finalizar esta unidad, contarás con un entorno de trabajo funcional para comenzar a programar en Python con mayor autonomía. Podrás crear y ejecutar scripts simples, interpretar la salida en la terminal, comprender el recorrido básico entre editor, intérprete y sistema operativo, y realizar correcciones iniciales frente a errores frecuentes. Esta base será necesaria para avanzar con seguridad hacia variables, operadores, interacción y resolución de problemas más elaborados.
