Este glosario reúne términos básicos que aparecen a lo largo de la unidad. Su propósito es facilitar la lectura de los documentos y ofrecer definiciones breves, claras y útiles para comenzar a trabajar con Python.

:::note[Uso recomendado]
Consulta este glosario cuando encuentres una palabra técnica que no recuerdes o quieras revisar con mayor precisión.
:::

## Lenguaje y ejecución

| Término | Definición |
|---|---|
| **Python** | Lenguaje de programación de alto nivel, interpretado y multipropósito. |
| **Intérprete** | Programa que lee y ejecuta instrucciones escritas en Python. |
| **Script** | Archivo de texto que contiene instrucciones escritas en Python. |
| **Programa** | Conjunto de instrucciones que una computadora puede ejecutar para realizar una tarea. |
| **Código fuente** | Texto escrito por una persona programadora en un lenguaje de programación. |
| **Ejecución** | Proceso mediante el cual el intérprete lee un programa y realiza sus instrucciones. |
| **Salida** | Resultado que un programa muestra en pantalla o entrega al sistema. |

## Archivos y organización

| Término | Definición |
|---|---|
| **Archivo** | Unidad de información guardada en el sistema, identificada por un nombre. |
| **Carpeta** | Contenedor que permite organizar archivos y otras carpetas. |
| **Ruta** | Ubicación de un archivo o carpeta dentro del sistema. |
| **Extensión** | Parte final del nombre de un archivo que indica su tipo. En Python se usa `.py`. |
| **Archivo `.py`** | Archivo de Python que contiene instrucciones escritas en este lenguaje. |
| **Directorio actual** | Carpeta en la que se encuentra ubicada la terminal en un momento determinado. |

## Entorno de trabajo

| Término | Definición |
|---|---|
| **Entorno de trabajo** | Conjunto de herramientas y configuraciones necesarias para escribir, ejecutar y revisar programas. |
| **Visual Studio Code** | Editor de código utilizado para escribir, organizar y ejecutar archivos de Python. |
| **Editor de código** | Programa diseñado para escribir y modificar código fuente. |
| **Extensión** | Complemento que agrega funciones a un editor como Visual Studio Code. |
| **Terminal** | Interfaz basada en texto que permite ejecutar comandos y programas. |
| **Consola** | Término que suele usarse como sinónimo de terminal, según el contexto. |
| **Homebrew** | Gestor de paquetes utilizado en macOS para instalar herramientas desde la terminal. |
| **Gestor de paquetes** | Herramienta que permite instalar, actualizar o eliminar software mediante comandos. |

## Comandos y configuración

| Término | Definición |
|---|---|
| **Comando** | Instrucción escrita en la terminal para solicitar una acción al sistema. |
| **PATH** | Variable del sistema que indica dónde buscar programas ejecutables. |
| **`python` / `python3`** | Comandos utilizados para ejecutar Python desde la terminal, según el sistema operativo. |
| **`python --version`** | Comando que muestra la versión instalada de Python. |
| **`python3 --version`** | Variante habitual del comando anterior en macOS y Linux. |
| **`quit()`** | Instrucción que permite salir del intérprete interactivo de Python. |

## Escritura del código

| Término | Definición |
|---|---|
| **Sintaxis** | Conjunto de reglas que indica cómo debe escribirse un programa para que sea válido. |
| **Instrucción** | Acción individual escrita en código para que el programa realice una tarea. |
| **Función** | Bloque reutilizable de código que realiza una acción específica. |
| **`print()`** | Función que muestra información en pantalla. |
| **Paréntesis** | Símbolos `(` y `)` usados, entre otras cosas, para llamar funciones. |
| **Comillas** | Símbolos usados para delimitar texto, por ejemplo `"Hola"` o `'Hola'`. |
| **Indentación** | Espacios al inicio de una línea que en Python ayudan a definir bloques de código. |

## Errores frecuentes

| Término | Definición |
|---|---|
| **Error** | Situación en la que el programa no puede ejecutarse correctamente o no produce el resultado esperado. |
| **Error de sintaxis** | Error producido por una instrucción mal escrita. |
| **Error de ejecución** | Error que ocurre cuando el programa intenta ejecutarse en una condición incorrecta. |
| **Error lógico** | Error en el que el programa se ejecuta, pero el resultado no es el esperado. |
| **Mensaje de error** | Texto que entrega Python o la terminal para indicar un problema. |
